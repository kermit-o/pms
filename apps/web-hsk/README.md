# @pms/web-hsk

Next.js 15 PWA — Housekeeping mobile-first.

> **Pendiente de scaffolding (Sprint 4).**

## Responsabilidades

- PWA instalable en móvil de gama media (target: Android 10+).
- Estado de habitaciones (Clean / Dirty / Inspected / OOO / OOS).
- Asignación de tareas a camareras.
- Discrepancies, Lost & Found.
- Funcionamiento offline-first (la conectividad WiFi en hoteles es irregular).
- i18n: ES + EN.

## Restricción no negociable

Si no funciona fluido en un móvil de gama media con conexión 3G, no se mergea.
