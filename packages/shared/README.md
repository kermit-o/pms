# @pms/shared

Tipos, schemas (Zod) y utilidades compartidos entre `api`, `web-fo` y `web-hsk`.

## Contenido previsto

- Domain types (Reservation, Guest, Room, Folio, etc.).
- Zod schemas para validación end-to-end.
- Utilidades de fechas/horarios (timezone-aware, business date logic).
- Utilidades de moneda (multi-divisa).
- Constantes de dominio (estados de habitación, tipos de cargo, etc.).
